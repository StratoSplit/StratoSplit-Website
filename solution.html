Envisioned Solution

Our solution is a mobile android application that will enable the tactical communication between a user and the radio modem by General Dynamics Mission Systems. This will enable users to more easily send and receive messages on a mobile device.

Our system architecture consists of the following:

  • A wifi-enabled android device that is able to communicate with the software defined radio (SDR) using a socket connection. This is the overarching systems diagram, however a radio simulator will be used by our team in place of the actual SDR.

  • Communication between the mobile device and the SDR will be simulated. The radio simulator is implemented in a linux environment running in VirtualBox. The android application will communicate through the RCI interface via a socket connection using an IP or Wifi connection.

Currently, a prototype has been developed that meets most of the user interface requirements for the solution. Users are able to navigate through the application, draft messages, and access the inbox. Communication with the radio simulator is still under development.
